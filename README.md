¿Qué es AgroEco IA?
AgroEco IA es una aplicación de análisis agroecológico que integra:
🔬 Lectura de cromatogramas de suelo (método Pfeiffer)
📊 Base de datos real con 150 muestras y 33 variables globales
🤖 Motor de recomendaciones basado en K-Nearest Neighbors
🌿 Recomendaciones contextualizadas por altitud, cultivo y tipo de manejo
